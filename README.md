# IMall

一个仿天猫APP的商城应用，基于ionic2框架开发,项目持续更新中~~~
目前有选购，注册，编辑等基本功能，有底部加载，登录缓存，阅读模式等交互功能。

## 安装

### 安装 Ionic

```bash
$ npm install -g ionic
```

进入项目目录，安装依赖，运行'ionic serve'命令来启动应用：

```bash
$ cd IMall-master
$ npm install
$ ionic serve
```
